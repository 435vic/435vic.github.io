This is the mandelbrot set.

Nice, isn't it? It is a set of complex numbers which are in a plane, and once put through a function, if the number, iterated enough, tends to go towards infinity (or negative infinity), then it is not part of the set.

Please note, I am massively simplificating, since I don't even understand it fully. I highly suggest investigating other sources if you want to know more. If you want, you can start here!
